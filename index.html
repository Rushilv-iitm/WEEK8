Example: Computing Quarterly Returns


def quarterly_return(start_price, end_price, dividends=0.0):
    """
    Simple quarterly return calculation.

    R_q = (P_end - P_start + D) / P_start
    """
    return (end_price - start_price + dividends) / start_price

# Example: stock went from 100 to 112 with 1 unit dividend
rq = quarterly_return(100, 112, dividends=1)
print(f"Quarterly return: {rq:.2%}")

NPV and Discounted Cash Flows

Net Present Value (NPV) of a project with cash flows \( CF_t \) and discount rate \( r \):

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} - CF_0 \]

Internal Rate of Return (IRR) is the rate \( r^* \) such that:

\[ 0 = \sum_{t=1}^{T} \frac{CF_t}{(1 + r^*)^t} - CF_0 \]

Relate these formulas to capital budgeting decisions and treasury functions in financial institutions.
